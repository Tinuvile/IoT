布局说明

这是IoT数据订阅系统的基础布局页面，包含以下组件：

  • 顶部导航栏 - 显示应用名称和当前时间
  • 侧边栏导航 - 提供各页面间的快速切换
  • 主内容区域 - 显示各页面的具体内容
  • 响应式设计 - 支持桌面端和移动端设备
主要功能页面：
  • 首页 - 系统概览和订阅状态
  • 数据订阅 - 订阅控制和配置管理
  • 数据监控 - 实时数据监控和分析结果
  • 系统布局 - 演示系统组件和样式

系统架构和技术栈展示
IoT数据订阅系统 - 技术架构展示
=====================================
应用类型: Spring Boot 订阅者应用
当前时间:
前端框架: Bootstrap 5 + 自定义CSS
图标库: Font Awesome 6
图表库: Chart.js (用于数据可视化)
WebSocket: SockJS + STOMP (实时通信)
=====================================
后端技术栈:
- Spring Boot 3.x (主框架)
- Spring Data JPA (数据持久化)
- MySQL Database (数据存储)
- Eclipse Paho MQTT (消息订阅)
- SLF4J + Logback (日志系统)
=====================================
主要功能模块:
- MQTT消息订阅服务
- 数据存储和管理服务
- 数据分析和统计服务
- WebSocket实时通信服务
- RESTful API接口服务
=====================================
支持的传感器类型:
- 温度传感器 (Temperature)
- 湿度传感器 (Humidity)
- 气压传感器 (Pressure)
=====================================
数据处理特性:
- 实时数据订阅和存储
- 统计分析 (平均值、最值、标准差)
- 数据质量评估
- 历史数据查询
- 数据导出功能
